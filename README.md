### About

I build production-grade web, mobile, and iOS applications end-to-end — from architecture and backend infrastructure to polished user interfaces. I run **TunaSoft**, a full-stack software agency, and ship commercial products across real estate, e-commerce, gaming, and AI tooling.

### What I work on

- **Web & backend** — TypeScript, Node.js, Laravel, .NET, Python, PostgreSQL, Redis, Docker
- **iOS** — Swift, SwiftUI, SceneKit, Clean Architecture, Coordinator pattern
- **Cross-platform mobile** — Flutter, React Native (Expo)
- **Real-time systems** — SignalR, WebSockets, live auctions, multiplayer game servers
- **AI products** — full-stack AI application builders and LLM-powered tooling
