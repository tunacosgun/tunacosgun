## 🏢 About

I run **TunaSoft Yazılım** with **Duygu Durmuş**, delivering commercial software across real estate, fintech, e-commerce, gaming, AI tooling, and privacy infrastructure. I ship complete products — architecture, backend, mobile, web, and deployment — with a strong bias toward clean domain modeling, operational robustness, and long-term maintainability.

### Core competencies

- **Go backend** — modular monoliths, DDD, clean architecture, event-driven systems, webhook idempotency
- **iOS / macOS** — Swift, SwiftUI, Network Extension, SceneKit, Clean Architecture, Coordinator pattern
- **Cross-platform mobile** — Flutter (iOS · Android · macOS · Desktop), React Native + Expo
- **Payments & subscriptions** — Stripe, iyzico, webhook idempotency, device binding, recurring state machines
- **VPN & networking** — WireGuard, VLESS / XRay, sing-box, Network Extension, Android VpnService
- **AI & LLM** — agent systems, RAG pipelines, LLM orchestration, semantic search
- **Real-time systems** — SignalR, WebSockets, multiplayer servers, live auctions
- **Admin panels & internal tools** — React, TypeScript, TanStack Query, low-code platforms
